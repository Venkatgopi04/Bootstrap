Bootstrap:
=======
->Bootstrap is a FrontEnd Framework meant for Responsive web applications

->Complete Bootstrap depends on classes and plugin(CDN)

->It is a HTML & CSS & JavaScript based Framework.

->Grid System:
=========
Bootstrap supports Grid System.It is process or methodlogy to develop the page layouts through 
the series of 12 rows and 12 column


Bootsrap classes with their device name:
==========================

-----------------------------------------------------------
Device_Name                                                       Bootstrap classes
-----------------------------------------------------------
For Extral small device                                             col-xs* 
For Small device                                                       col-sm* 
For Medium device                                                   col-md* 
For Large device                                                       col-lg*
-------------------------------------------------------------


Note:
-----
       In Boostrap framework width will automatically used by bootstrap itself.
